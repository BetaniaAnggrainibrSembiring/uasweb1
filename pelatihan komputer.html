BERITA DESA







Perangkat Desa Harapan Mengikuti Program Pelatihan Komputer Dasar di Universitas Dehasen Bengkulu




Alhamdulillah, pada tanggal 27 November 2021 Perangkat Desa Harapan termasuk juga BPD, Pengurus BUMDes dan beberapa pegurus lembaga masyarakat Desa Harapan berkesempatan untuk mengikuti Program Pelatihan Gratis di Universitas Dehasen Bengkulu. Dalam pelatihan tersebut diajarkan bagaimana mengelola file dan folder (manajemen file) dengan Explorer Windows, membuat berbagai macam dokumen menggunakan Microsoft Word termasuk pembuatan surat dan dokumen-dokumen lainnya. Microsoft Excel juga diajarkan untuk pembuatan data tabular dan perhitungan, seperti perhitungan keuangan dan lainnya. Selain itu di sesi akhir diajarkan cara membuat slide presentasi menggunakan Microsoft PowerPoint.
